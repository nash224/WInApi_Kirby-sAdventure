= 비트맵 설명 =
다음은 비트맵 설명이다.
// 청록색 비트맵 : 청록색 비트맵을  벗어나게 되면   떨어진다

// 초록 비트맵 : 초록색 비트맵은  레벨 이동이 가능하다 => *DebugMode Off => 첫번째 스테이지로 입장  하겠다
=> Level End 



1 스테이지 진행 
Start Level 
1. 이 몬스터는   일반 몬스터 Waddle Dee 이다 => 이 몬스터를  커비의 기본 특수능력으로 먹으면  Contain상태가 된다 
     => 일반 몬스터를 먹으면  아무능력도 얻지 못한다.

2. Waddle Doo : Waddle Doo는 일반 몬스터과 다르게   특수한 능력을 가지고 있다
   => 이 몬스터를 삼키면 특수 능력을 얻을 수 있다
   => 특수 모드가 있는 커비와   몬스터가 충돌을 하게 되면   커비의 피가 줄어들고   모드의 정보가 들어있는 별을 떨어뜨리게 된다.

3. Sparky : *BitMap Mode On => Sparky는 계단을 인식할 수 있어   계단을 뛰어 올라갈 수 있다.
   => 떨어진 별은 다시 먹을 수 있고   커비가 잃어버린 모드를 되찾을 수 있다.

4. Sword Knight : Sword Knight는  커비가 다가오면   커비를 향해 검을 휘두른다. => 커비의 날숨으로 잡을 수 있다

5. 피가 없다면 Drink를 먹어 HP를 회복할 수 있다. 

6. Hot Head : 2가지 패턴 => 하나는 실선보다 멀 때 화염구를 쏘지만   각도 제한이 있어   높이 날고 있는 커비를 맞추지 못한다  
    => 다른 하나는   커비가 실선보다 가까우면   브레스를 내뿜게 된다 => 몬스터의 능력을 얻고   다음 필드로 이동  하겠다 => *FireMode Enter => End Level



2 스테이지 진행 
Start Level : 커비의 모드가 있으면   레벨 이동을 해도   모드 유지가  가능하다 
              => 여기서부턴 빠른 진행을 위해   충돌 처리를 끄고    진행   하겠다 => // 개발자 모드 Off

1. 뛰어다니는 난쟁이는 일반 몬스터인 PoppyBros & Jr 이다 => Kabu는  한번씩 높이 뛸 수 있다. 이 몬스터도 일반 몬스터다 
   => 바닥을 쓸고 있는   Broom Hatter 도 일반 몬스터이다

2. Sword Knight : 이전 필드에서 보여드린 몬스터이다. =>  이번엔 모드을 얻기 위해   삼켜보겠다.

3. Togezo : 토게조는 커비를 인식했을 때   앞으로 구르다가   절벽에서 멈추고   일정 시간이 지나면 방향을 반전시켜 커비를 공격한다
   => // 개발자 모드 On

4. 이 비행 몬스터는 Scarfy이다. => 평상시에는 공격 하지 않고 커비를 바라볼 뿐이지만
   => 커비가 Scarfy를 먹을려고 하면    돌변 하면서   커비에게 다가가 몸을 터트린다.

5. Laser Ball : Laser Ball은 하늘을 날아다니는 파워 몬스터이다 => 디버깅 네모 영역을 기준으로  가까우면 도망가고  멀어지면 쫗아온다
   => 그러다 커비가  네모 영역에 있으면   몬스터는 커버에게 레이저를 발사한다 
   => // 개발자 모드 해제
  모드 해제 => End Level



Hub
LevelStart => 한 스테이지가 끝나면    커비가 문을 열고   스테이지를 클리어 했다는 증표로    문앞에 깃발을 꽂는다 
           => 이어서 보스 몬스터를 잡기 위해   보너스 맵에서 능력을 얻고   보스 스테이지로 입장 하겠다 => Level End

보스 패턴
LevelStart => 커비가 특정 범위에 들어서게 되면    보스 HP가 차오르면서    보스 스테이지가 시작된다. 

1. 보스 몬스터는 사과를 떨어트리거나  휫바람 패턴이 있다 => // 사과 4번 떨어지고 => // 개발자모드 On => // 치트로 휫바람 유도 
   => 지금 커비의 모드는 Ice 모드 이다. => 몬스터를 얼려 얼음을 만들 수 있다 => 커비가 얼음에 닿으면 얼음을 날려버릴 수 있다 
   => 그리고 보스 몬스터에게  아이스 브레스로 직접  공격이 가능하다 => // 아이스블럭 생성 => // 모드 해제 
   => 아이스 블럭이 있는 상태에서   커비가 모드를 해제하게 되면   동시에 얼음이 사라지게 된다

2. 커비의 공격수단으로   입에 든 별을 발사할 수 있다 
   => 일반적으로 작은별은   보스 몬스터에게    모드로 공격하는 것 보다   대미지가 크게 들어간다
   => 몬스터를 2마리 이상 먹으면   큰별을 발사할 수 있게 되는데    작은별보다 대미지가 높게 들어간다

3. 보스 몬스터의 Hp가 모두 없어지면   보스 스테이지가 클리어 되고   보스 몬스터에서 막대사탕이 나오게 된다. 
   => 막대 사탕을 먹으면   커비가 춤을 추면서   게임이 끝나게 됩니다.
